| 定义        | 含义                                                         |
| :---------- | :----------------------------------------------------------- |
| int *p      | p的含义:指针变量                                             |
| int **p     | p的含义:二级指针,指向int * 类型的数据                        |
| int *p[n]   | p的含义:指针数组.[ ] 的优先级高于 *，可以理解为(int *(p[n])) |
| int (*p)[n] | p的含义:二维数组指针                                         |
| int *p()    | p的含义:一个返回值类型为int *的函数                          |
| int (*p)()  | p的含义:函数指针,指向原型为int func() 的函数                 |
